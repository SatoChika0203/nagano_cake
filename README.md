# Nagano cake

こちらのサイトは、DMMWEBCAMPの応用知識学習フェーズで作りました。
試行錯誤し、なんとか形になりましたが、まだまだ拙い記述ばかりです。
暖かい目でご覧いただければ幸いです。

## 実装機能
Costomer(ユーザー側)
- 商品の確認、注文処理
- 注文履歴の確認
- 会員情報の確認、編集、退会

Admin(管理者側)
- 商品の確認、追加、編集
- 注文履歴の確認
- 会員情報の確認、編集

## 開発環境
- Ruby 3.1.2
- Rails 6.1.7.3
- HTML5
- Gem
  - Bootstrap4
  - devies
